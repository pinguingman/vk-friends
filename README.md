# vk-friends
Site based on flask, uses vk oauth to get list of user friends.

## Requirements:
- python module `flask`
- python module `requests`
- vk app `Application ID`, `Secure key` and `Service token`

## Run:
###### on Windows
    set FLASK_APP=main.py
    flask run
###### on nix
    FLASK_APP=main.py flask run
